# Legacy Drivechain Docs

The files in this directory describe the legacy drivechain design and its
associated operational material.

They are retained only as migration context for the current repository state.
They are not the target architecture for new work.

The trustless redesign target is documented in:

- [../validity-sidechains/LIP-validity-sidechains.md](../validity-sidechains/LIP-validity-sidechains.md)
- [../validity-sidechains/MIGRATION_MAP.md](../validity-sidechains/MIGRATION_MAP.md)
- [../validity-sidechains/ACTIVATION_REQUIREMENTS.md](../validity-sidechains/ACTIVATION_REQUIREMENTS.md)

Until the consensus, RPC, wallet, and test code is migrated away from
drivechain, these legacy documents remain useful reference material but should
not be treated as the end-state protocol specification.
